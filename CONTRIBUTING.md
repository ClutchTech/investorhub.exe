# Contributing

This is a private/proprietary Clutch Systems HQ investor website.

Contributions are only accepted from authorized collaborators approved by Anthony Tanksley.

## Development Rules

- Keep branding premium, sharp, and investor-ready.
- Do not remove Clutch ownership language.
- Do not expose private financial documents.
- Do not commit `.env` files or secret keys.
- Keep investor metrics easy to edit in a single content/config area.
- Keep QR business cards mobile-first and print-friendly.

## Commit Style

```bash
git commit -m "Add investor deck hero section"
git commit -m "Update Clutch product suite cards"
git commit -m "Add QR business card layout"
git commit -m "Prepare Vercel deployment config"
```
